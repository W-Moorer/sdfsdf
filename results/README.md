# Results Directory

本目录用于存放投稿实验的统一输出结果。

## 约定

- 原始数据使用 `CSV`
- 图表使用 `PNG`
- 文件名应与算例和指标对应

## 推荐命名

- `cylinder_omega.csv`
- `cylinder_omega_decay.png`
- `sharp_penetration.csv`
- `sharp_comparison.png`
- `mass_ratio_residual.csv`
- `mass_ratio_convergence.png`
- `mass_ratio_energy.csv`
- `summary_table.csv`

## 要求

- 图表必须可由仓库脚本从 CSV 自动生成
- 每个结果文件都必须能回指到具体实验参数和命令
